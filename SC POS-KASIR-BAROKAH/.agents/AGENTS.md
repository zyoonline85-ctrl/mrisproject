# Project Rules — POS Barokah Kasir Mobile App

- **Siklus Versi APK**: Setiap kali melakukan kompilasi atau deploy APK baru, wajib menaikkan nomor versi (`version`) di berkas `pubspec.yaml` (baik `versionName` maupun `versionCode`, contoh: dari `0.1.0+1` ke `0.1.1+2` atau seterusnya) untuk memberikan pembeda yang jelas pada instalasi aplikasi kasir.
